## 🐾 API de Animais

Projeto em Python que consome a API Ninjas para buscar informações sobre animais e salvar favoritos localmente em JSON.

### 🚀 Tecnologias
- Python
- Requests
- JSON

### ⚙️ Como rodar
1. Clone o repositório
2. Instale as dependências: pip install requests
3. Execute: python main.py

### 💡 Funcionalidades
- Buscar animais por nome (em inglês)
- Ver dieta, habitat e tempo de vida
- Salvar animais favoritos
- Listar favoritos salvos
